# Blog

Personal blog of interesting technical achievements.

The blog is hosted at [http://www.jonathanpearlin.com](http://www.jonathanpearlin.com).

## Build and Deploy

### Build

```sh
> ./gradlew clean build
```

### Local Testing

```sh
> ./gradlew jettyRun
```

### Deploy

```sh
> ./gradlew clean build publish
```
